## [0.2.0] - 2022-09-12
### Changed
- The `On OSC Message` OmniGraph node now outputs a Bundle typed value rather than an Unknown typed value.
    - Users can extract the "address" and the "arguments" of the OSC message with the `Extract Attribute` node.
